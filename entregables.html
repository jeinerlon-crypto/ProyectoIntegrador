Entregables

Un entregable es el producto final o parcial que se presenta al finalizar una actividad, práctica o proyecto. Es la evidencia del trabajo realizado y debe mostrar de manera clara qué se hizo, cómo se hizo y con qué resultados.

Características principales de un entregable

Tiene un propósito definido: demostrar el avance o cumplimiento de un objetivo (por ejemplo, diseñar una página web o crear una base de datos). Está bien estructurado: incluye portada, desarrollo, conclusiones o resultados. Es claro y ordenado: usa títulos, subtítulos y descripciones para que sea fácil de entender. Incluye evidencias: puede contener texto, capturas de pantalla, código o enlaces. Tiene formato uniforme: puede ser un documento (Word, PDF), una presentación o un archivo HTML.
